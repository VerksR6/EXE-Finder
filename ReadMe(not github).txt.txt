 Verks’ EXE Finder

README / How to Use

What Is Verks’ EXE Finder?

Verks’ EXE Finder is a batch (.bat) tool made to help you find .exe files on your PC, such as games and applications.

✅ It does NOT scan hardware
✅ It does NOT damage your PC
✅ It is NOT a virus

This tool is only meant to make it easier to locate game and app executables without messing up your system.

How to Use

Open Verks' EXE Finder.bat

You will see two options:

Press 1 → Start scanning

Press 2 → Close the program

Press 1 to proceed.

The program will display “Scanning files and apps”

It scans user files, games, and applications

It does NOT scan PC or hardware files

When the scan is finished:

A .txt file will automatically open

This file contains a list of all detected .exe files

Press CTRL + F in the text file:

Type the name of the game or app

This lets you quickly check if you have the .exe

Close the text file and return to the batch window.

Press any key to go back to the main home screen.

If It Doesn’t Work

Try running the file as Administrator

Right-click the .bat file

Click Run as administrator

Some systems require admin access to scan certain folders.

Is This a Virus?

❌ NO.
This is not a virus, malware, or spyware.

✔ It only scans file names
✔ It does not delete or modify files
✔ It does not run any executables

It is simply a useful tool to help you find games and app EXE files easily.
